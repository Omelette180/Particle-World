# Particle-World

Particle-World is a sandbox simulation game where physics, chemistry, ecosystems, and city systems interact in real time. Players can draw elements directly into the world and watch them react with gravity, temperature, and other materials.

## Play

https://omelette180.github.io/Particle-World/


## Controls

Left Click — Draw element
Right Click — Erase
Scroll Wheel / Arrow Keys — Change brush size
Space — Pause simulation
. (Dot) — Step one frame
/ or - — Change simulation speed
H — Toggle heat map

## Features

### Physics System

* Materials have weight and fall at different speeds
* Heat spreads differently through metal, wood, and rubber
* Ice melts into water and water turns into steam
* Overheated batteries can explode

### Ocean Ecosystem

* Seawater, coral, and kelp create marine environments
* Sharks hunt fish and crabs
* Whales drift through the ocean and release bubbles
* Radiation harms marine life

### Chemistry Lab

* Combine chemicals to trigger reactions
* A + B → Fire
* A + C → Acid
* B + C → Steam
* A + B + C → Nuclear reaction
* Catalysts increase reaction speed

### City Builder

* Build roads, houses, shops, and offices
* Citizens follow daily routines (home → work → shop → sleep)
* Emergency services respond to disasters

### Events

* Zombie outbreaks can invade the city
* Alien UFOs can abduct citizens

### Technology

* Logic gates (AND / OR / NOT / Timer)
* Portals teleport particles between locations
* Wormholes eject particles to random edges of the world

## Project

Particle-World is an experimental sandbox designed to explore emergent simulation systems where many mechanics interact at once.
